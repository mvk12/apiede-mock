### Instalación

1. Clone repository
   ```sh
   git clone git@github.com:mvk12/apiede-mock.git
   ```

2. Instalar dependencias composer
   ```sh
   composer install
   ```

3. Create `.env` file
   ```sh
   cp .env.example .env
   ```

4. Run app

   ```sh
   php -S localhost:8080 -t public
   ```
